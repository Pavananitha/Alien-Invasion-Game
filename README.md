# 👾 Alien Invasion - Desktop Game using Pygame

A classic-style 2D shooting game built using Python and Pygame. The player controls a spaceship that can shoot bullets to destroy aliens and try to survive as long as possible.

## 🖥️ Type of Application

This is a **desktop application**, not a web or cloud-deployable app. It is meant to be run on a local machine with Python and Pygame installed.

## 🎮 Game Features

- Spaceship movement and shooting
- Multiple aliens spawning in formation
- Bullet-alien collision detection
- Score tracking
- Difficulty progression
- Start and Game Over buttons

## 🚀 Getting Started

### Prerequisites

- Python 3.x
- Pygame

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/your-username/alien-invasion.git
   cd alien-invasion


2. Install required library:
   pip install pygame

3. Run the game:
   python alien_invasion.py
